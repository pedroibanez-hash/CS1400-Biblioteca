Lógica Booleana en Python
Este proyecto es una breve práctica en Python diseñada para entender y aplicar operadores booleanos como and, or y not. A través de ejemplos simples y condiciones realistas, aprenderás cómo construir expresiones lógicas que toman decisiones en tu código.

Objetivo: El objetivo de esta práctica es reforzar el uso de operadores lógicos en Python mediante ejercicios con variables como edad, permisos y días de la semana.

# Variables iniciales
El programa parte de tres variables ya definidas:

edad = 16
tiene_permiso = True
es_finde = False

Estas variables simulan un escenario donde queremos saber si una persona puede salir sola, dependiendo de su edad, si tiene permiso, y si es fin de semana.
