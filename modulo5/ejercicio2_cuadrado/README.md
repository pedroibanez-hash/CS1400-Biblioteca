#  ¿Es un Cuadrado Perfecto?

Este programa te ayuda a determinar si un número es un cuadrado perfecto usando Python.

##  Objetivos

- Practicar funciones (`def`)
- Usar condicionales (`if`, `else`)
- Trabajar con entradas (`input`)
- Realizar pruebas automáticas con listas y `assert`

##  Instrucciones

1. Completa la función `es_cuadrado_perfecto(n)` siguiendo las pistas.
2. Ejecuta el programa.
3. Introduce cualquier número y revisa si es un cuadrado perfecto.
4. Observa las pruebas automáticas al final del programa para más ejemplos.

##  ¿Qué es un cuadrado perfecto?

Un número es cuadrado perfecto si existe un número entero que, al multiplicarse por sí mismo, da como resultado el número original.

**Ejemplos:**

- `4` es un cuadrado perfecto porque `2 * 2 = 4`
- `9` es un cuadrado perfecto porque `3 * 3 = 9`
- `8` **NO** es un cuadrado perfecto
